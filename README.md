# Car Rental System

A full-stack Java web application built using:
- Java (JSP, Servlets)
- JDBC
- MySQL
- Apache Tomcat
- HTML, CSS

## Features
-  register login & dashboard
- Add, view, update cars
- User login & booking system


## Technologies Used
- Java
- JSP & Servlets
- MySQL
- HTML / CSS
- Apache Tomcat 9

## Runing step
1. Import project into Eclipse
2. Configure Tomcat Server
3. Create MySQL database & tables
4. Run on Server
